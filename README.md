# Commands
+------------+---------------------------------------+
| clear      |  Clears the screen!                   |
+------------+---------------------------------------+
| exit       |  Exits xTools :/                      |
+------------+---------------------------------------+
| help       |  Shows this dialog                    |
+------------+---------------------------------------+
| open tool  |  Lets you choose what tool too open   |
+------------+---------------------------------------+
| tools      |  Shows the list of tools              |
+------------+---------------------------------------+
| info       |  Shows credits for xTools             |
+------------+---------------------------------------+
| bash       |  Opens a bash session to run commands |
+------------+---------------------------------------+
